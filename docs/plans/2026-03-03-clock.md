# Clock Feature Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Render a centered green digital clock (HH:MM:SS) with chromatic aberration, brightness flicker, and occasional digit glitch over the matrix rain.

**Architecture:** A new `ClockRenderer` in `src/render/clock.rs` owns a cached SDL2 texture of the time string, rebuilds it once per second, and draws three passes (red ghost offset left, blue ghost offset right, green main with sine flicker). It is called from the main render loop in `src/render/mod.rs` after drawing columns, on the primary display only.

**Tech Stack:** Rust, SDL2 (ttf feature already enabled), `chrono` crate for local time, `rand` (already in Cargo.toml).

### Task 3: Implement `ClockRenderer::render` fully

**Files:**
- Modify: `src/render/clock.rs`

Replace the entire file with the full implementation:

```rust
use sdl2::render::{Canvas, Texture, TextureCreator};
use sdl2::video::{Window, WindowContext};
use sdl2::pixels::Color;
use sdl2::rect::Rect;
use sdl2::ttf::Font;
use rand::Rng;
use chrono::Local;

pub struct CachedClockTexture<'a> {
    pub texture: Texture<'a>,
    pub w: u32,
    pub h: u32,
}

pub struct ClockRenderer {
    last_second: u32,
}

impl ClockRenderer {
    pub fn new() -> Self {
        ClockRenderer { last_second: u32::MAX }
    }

    pub fn render<'a>(
        &mut self,
        canvas: &mut Canvas<Window>,
        texture_creator: &'a TextureCreator<WindowContext>,
        font: &Font,
        elapsed_secs: f32,
        rng: &mut impl Rng,
        cache: &mut Option<CachedClockTexture<'a>>,
    ) -> anyhow::Result<()> {
        let now = Local::now();
        let current_second = now.second();

        // Rebuild texture when the second changes
        if current_second != self.last_second || cache.is_none() {
            let time_str = now.format("%H:%M:%S").to_string();
            let surface = font
                .render(&time_str)
                .blended(Color::RGB(255, 255, 255))
                .map_err(|e| anyhow::anyhow!(e))?;
            let texture = texture_creator
                .create_texture_from_surface(&surface)
                .map_err(|e| anyhow::anyhow!(e))?;
            let q = texture.query();
            *cache = Some(CachedClockTexture { texture, w: q.width, h: q.height });
            self.last_second = current_second;
        }

        let Some(cached) = cache else { return Ok(()); };

        // Glitch: 0.3% chance per frame — replace one char with random katakana
        let display_str = if rng.gen::<f32>() < 0.003 {
            let time_str = now.format("%H:%M:%S").to_string();
            let mut chars: Vec<char> = time_str.chars().collect();
            // Only glitch digit positions (0,1,3,4,6,7)
            let digit_positions = [0usize, 1, 3, 4, 6, 7];
            let pos = digit_positions[rng.gen_range(0..digit_positions.len())];
            let katakana: Vec<char> = (0xFF66u32..=0xFF9F).filter_map(char::from_u32).collect();
            chars[pos] = katakana[rng.gen_range(0..katakana.len())];
            Some(chars.into_iter().collect::<String>())
        } else {
            None
        };

        // If glitching, rebuild a one-shot texture for this frame
        let (tex_ref, tw, th) = if let Some(glitch_str) = &display_str {
            let surface = font
                .render(glitch_str)
                .blended(Color::RGB(255, 255, 255))
                .map_err(|e| anyhow::anyhow!(e))?;
            let glitch_tex = texture_creator
                .create_texture_from_surface(&surface)
                .map_err(|e| anyhow::anyhow!(e))?;
            let q = glitch_tex.query();
            // We need to render it immediately since it's a local texture
            let (sw, sh) = canvas.output_size().unwrap_or((1920, 1080));
            let cx = (sw as i32 - q.width as i32) / 2;
            let cy = (sh as i32 - q.height as i32) / 2;

            // Red ghost
            let mut t = texture_creator
                .create_texture_from_surface(
                    font.render(glitch_str).blended(Color::RGB(255,255,255))
                        .map_err(|e| anyhow::anyhow!(e))?
                )
                .map_err(|e| anyhow::anyhow!(e))?;
            t.set_color_mod(180, 0, 0);
            t.set_alpha_mod(180);
            let _ = canvas.copy(&t, None, Some(Rect::new(cx - 3, cy, q.width, q.height)));

            // Blue ghost
            let mut t2 = texture_creator
                .create_texture_from_surface(
                    font.render(glitch_str).blended(Color::RGB(255,255,255))
                        .map_err(|e| anyhow::anyhow!(e))?
                )
                .map_err(|e| anyhow::anyhow!(e))?;
            t2.set_color_mod(0, 0, 180);
            t2.set_alpha_mod(180);
            let _ = canvas.copy(&t2, None, Some(Rect::new(cx + 3, cy, q.width, q.height)));

            // Green main
            let g = ((elapsed_secs * 2.0).sin().abs() * 30.0 + 225.0) as u8;
            glitch_tex.set_color_mod(0, g, 0);
            let _ = canvas.copy(&glitch_tex, None, Some(Rect::new(cx, cy, q.width, q.height)));

            return Ok(());
        } else {
            (&cached.texture, cached.w, cached.h)
        };

        let (sw, sh) = canvas.output_size().unwrap_or((1920, 1080));
        let cx = (sw as i32 - tw as i32) / 2;
        let cy = (sh as i32 - th as i32) / 2;

        // Red ghost
        tex_ref.set_color_mod(180, 0, 0);
        tex_ref.set_alpha_mod(180);
        canvas.copy(tex_ref, None, Some(Rect::new(cx - 3, cy, tw, th)))
            .map_err(|e| anyhow::anyhow!(e))?;

        // Blue ghost
        tex_ref.set_color_mod(0, 0, 180);
        tex_ref.set_alpha_mod(180);
        canvas.copy(tex_ref, None, Some(Rect::new(cx + 3, cy, tw, th)))
            .map_err(|e| anyhow::anyhow!(e))?;

        // Green main with sine flicker
        let g = ((elapsed_secs * 2.0).sin().abs() * 30.0 + 225.0) as u8;
        tex_ref.set_color_mod(0, g, 0);
        tex_ref.set_alpha_mod(255);
        canvas.copy(tex_ref, None, Some(Rect::new(cx, cy, tw, th)))
            .map_err(|e| anyhow::anyhow!(e))?;

        Ok(())
    }
}
```

**Note on the glitch branch:** The glitch path creates one-shot textures locally. This is slightly wasteful but happens at most 0.3% of frames. If SDL2 texture creation becomes a bottleneck, cache a second slot — but YAGNI for now.
